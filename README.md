在线稿件管理

1.角色权限
```
（1）普通作者: 查看稿件 上传稿件 在线编辑 个人资料管理
（2）专家: 审查稿件 个人资料管理
（3）普通编辑: 分发稿件 个人资料管理
 (4) 主编： 确定稿件是否通过 个人资料管理 他人资料管理
 ```

 2.稿件
 ```
(1) 审核： 上传（普通作者）
	  --> 分发（普通编辑）
	  --> 审核（专家）
	  --> 定稿 (主编)
(2) 统计：按审核状态 时间 分页
(3) 在线编辑 上传 下载
```

3.个人资料
```
(1) 注册 注册时填写 选填 必填
(2) 登录
(3) 修改
```

4.应用技术
```
(1) yii 2.0
(2) antd
(3) phpword
```
